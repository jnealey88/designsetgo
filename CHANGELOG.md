# Changelog

All notable changes to the DesignSetGo plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-13

### Added

#### Core Plugin Features
- **Block Manager** - Enable/disable individual blocks and extensions
- **Global Styles Manager** - Centralized style configuration
- **GDPR Compliance Module** - Privacy-focused form handling with consent management
- **Custom CSS Renderer** - Support for block-level and global custom styles
- **Settings Panel** - Comprehensive plugin configuration interface

#### Container Blocks (3)
- **Section** - Full-width container with background and spacing controls
- **Grid** - Advanced CSS Grid layout with responsive columns
- **Row** - Flexible row container for content organization

#### Interactive Blocks (11)
- **Accordion** - Collapsible content panels with smooth animations
- **Scroll Accordion** - Scroll-triggered accordion behavior
- **Image Accordion** - Image-based accordion with hover effects
- **Tabs** - Tabbed content interface with multiple style options
- **Slider** - Content carousel with navigation and autoplay
- **Flip Card** - Interactive card with front/back flip animation
- **Scroll Marquee** - Auto-scrolling content ticker
- **Reveal** - Scroll-triggered content reveal animations
- **Countdown Timer** - Configurable countdown with multiple display formats
- **Counter** - Animated number counter with increment effects

#### Form System (13 Blocks)
- **Form Builder** - Complete form container with submission handling
- **Form Fields** - Text, Email, Textarea, Checkbox, Date, Time, Number, Phone, URL, Hidden, Select, Password
- **Security Features** - Honeypot protection, rate limiting, CSRF protection
- **Email Integration** - Automated form submission notifications

#### Styled & UI Blocks (8)
- **Icon** - 40+ built-in SVG icons with size and color controls
- **Icon Button** - Customizable button with icon support
- **Icon List** - Styled lists with icon markers
- **Pill** - Decorative pill/badge component
- **Progress Bar** - Animated progress indicators
- **Divider** - Content separator with multiple styles
- **Blobs** - Organic SVG blob shapes for backgrounds

#### Block Extensions (11)
- **Block Animations** - Entrance and exit animations for all blocks
- **Animation Controls** - Load-based animations via data attributes
- **Responsive Visibility** - Show/hide blocks by device (mobile/tablet/desktop)
- **Sticky Header** - Template part sticky positioning controls
- **Clickable Group** - Make any container block clickable
- **Background Video** - Video background support for containers
- **Custom CSS** - Add custom classes and inline styles to blocks
- **Grid Span** - Column spanning controls for Grid block children
- **Max Width** - Content width constraint controls
- **Reveal Controls** - Scroll-based reveal animation settings
- **Text Alignment Inheritance** - Child blocks inherit parent text alignment

#### Technical Features
- **WordPress 6.4+ Compatibility** - Modern Block Editor API support
- **FSE Ready** - Full Site Editing compatibility
- **Block Supports** - Comprehensive use of WordPress native block supports
- **Accessibility** - WCAG 2.1 compliant components
- **Performance Optimized** - Lightweight asset loading, conditional script enqueuing
- **Security Hardened** - Nonce validation, capability checks, sanitization
- **Internationalization** - Translation ready with .pot file
- **Developer Friendly** - Extensive hooks, filters, and documentation

#### Design System
- **WordPress Presets** - Integrates with theme color palettes and typography
- **Responsive Controls** - Mobile-first responsive design tools
- **Custom Properties** - CSS custom property support for dynamic styling
- **Layout Patterns** - Reusable design patterns and templates

### Requirements
- WordPress 6.0 or higher
- PHP 7.4 or higher
- Modern browser with JavaScript enabled
