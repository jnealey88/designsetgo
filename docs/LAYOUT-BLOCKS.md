# Layout Blocks

Build powerful, flexible page layouts with our suite of container blocks. Each block is designed to work seamlessly with WordPress's native layout system while providing enhanced control and features.

---

## Section

**Vertical stacking container for building page sections**

The Section block is your foundation for creating structured page layouts. Perfect for hero sections, content areas, and any vertical content organization. Sections span the full width of your page by default, with intelligent inner width constraints to keep your content readable and centered.

### Key Features

- **Full-width by default** with constrained inner content
- **Vertical stacking** with consistent spacing
- **Customizable width constraints** - toggle on/off or set custom max-width
- **Complete styling control** - padding, margins, colors, borders
- **Hover effects** - background color and text color transitions
- **Nested layouts** - combine with Row and Grid blocks for complex designs

### Best For

- Hero sections
- Content areas
- Feature sections
- Testimonial blocks
- Call-to-action sections

## Row

**Horizontal flex container for side-by-side layouts**

The Row block creates flexible horizontal layouts with automatic wrapping. Ideal for button groups, card layouts, feature lists, and any content that needs to flow horizontally with responsive behavior.

### Key Features

- **Horizontal layout** with flexible wrapping
- **Full-width support** - spans container width automatically
- **Mobile stack option** - converts to vertical layout on small screens
- **Flexible alignment** - left, center, right justification
- **Responsive gap control** - consistent spacing between items
- **Hover effects** - inherited from parent or custom settings

### Best For

- Button groups
- Feature cards
- Image galleries
- Icon lists
- Navigation elements
- Stats and metrics

## Grid

**Powerful CSS Grid container for complex layouts**

The Grid block leverages CSS Grid to create sophisticated multi-column layouts with precise control. Perfect for card layouts, galleries, and any design requiring structured grid-based organization.

### Key Features

- **Responsive columns** - define column counts for desktop, tablet, and mobile
- **Auto-fit and auto-fill** - intelligent column sizing
- **Gap control** - separate row and column gap settings
- **Flexible alignment** - both horizontal and vertical
- **Min/max column widths** - precise control over grid behavior
- **Dense packing** - optional auto-flow dense for optimal space usage

### Best For

- Card grids
- Product listings
- Image galleries
- Portfolio layouts
- Team member sections
- Feature showcases

## Layout Combinations

These blocks are designed to work together:

- **Section + Row** - Full-width section with centered horizontal content
- **Section + Grid** - Full-width section with multi-column grid layout
- **Row + Grid** - Horizontal layout containing grid-based cards
- **Nested Sections** - Complex page structures with multiple content areas

All layout blocks support WordPress's native block supports including spacing, colors, typography, borders, and more.
