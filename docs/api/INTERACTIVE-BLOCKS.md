# Interactive Blocks

Add engaging, interactive elements to your site with animations, user controls, and dynamic content. All blocks are fully accessible, responsive, and performant.

---

## Accordion

**Collapsible content sections for FAQs and content organization**

Create accessible accordion sections that expand and collapse to reveal content. Perfect for FAQs, feature lists, and organizing large amounts of content in a compact space.

### Key Features

- **Flexible behavior** - Allow single or multiple open panels
- **Initial state control** - Start with none, first, or all items open
- **Custom icons** - Choose from chevron, plus-minus, caret, or no icon
- **Icon positioning** - Place icons on left or right
- **Hover effects** - Custom hover colors for headers
- **Open state styling** - Different colors for active items
- **Keyboard navigation** - Full keyboard and screen reader support
- **Borders and spacing** - Control borders between items and spacing

### Best For

- FAQ sections
- Product specifications
- Feature lists
- Privacy policies
- Documentation
- Step-by-step guides

## Tabs

**Tabbed content with icons, deep linking, and responsive modes**

Organize content into tabbed panels with smooth transitions. Features horizontal and vertical orientations, icon support, and automatic conversion to accordion on mobile.

### Key Features

- **Orientation options** - Horizontal or vertical tab layout
- **Tab alignment** - Left, center, right, or justified
- **Multiple styles** - Default, pills, underline, or minimal
- **Icon support** - Add icons to tab labels
- **Deep linking** - Direct URLs to specific tabs
- **Mobile responsive** - Auto-converts to accordion or dropdown on mobile
- **Custom colors** - Full control over active and inactive states
- **Smooth transitions** - Animated content switching

### Best For

- Product features
- Pricing tables
- Portfolio categories
- Service descriptions
- Content comparison
- Multi-step processes

## Slider

**Modern carousel with multiple effects and auto-play**

Create engaging slideshows with various transition effects, navigation controls, and responsive behavior. Supports any WordPress blocks inside slides.

### Key Features

- **Multiple effects** - Slide, fade, cube, flip, and more
- **Responsive slides** - Different slide counts per device
- **Navigation controls** - Arrows and dot indicators with custom styling
- **Auto-play** - Automatic rotation with pause on hover
- **Loop mode** - Continuous infinite loop
- **Touch/swipe** - Mobile-friendly touch gestures
- **Free mode** - Momentum-based scrolling
- **Centered slides** - Highlight active slide in center
- **Custom height** - Fixed height or aspect ratio

### Best For

- Hero sections
- Image galleries
- Testimonial carousels
- Product showcases
- Featured content
