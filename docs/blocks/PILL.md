# Pill Block - User Guide

**Version**: 1.0.0
**Category**: Design
**Keywords**: pill, badge, tag, label

## Overview

The **Pill Block** displays text with a rounded background that wraps tightly around content. Perfect for tags, labels, badges, status indicators, categories, and inline highlights that need to stand out.

**Key Features:**
- Inline display (wraps naturally in content flow)
- Compact, rounded design
- Full WordPress Block Supports (color, typography, spacing, border)
- Rich text formatting (bold, italic)
- Minimal markup and lightweight

- Inline display (fits in text flow)

## ‚úÖ Best Practices

**DO:**
- Keep text short (1-3 words ideal)
- Use high contrast colors (text vs background)
- Use sparingly for emphasis
- Test readability at different screen sizes

**DON'T:**
- Use for long sentences or paragraphs
- Overuse on a single page (loses impact)
- Use low-contrast color combinations
- Rely solely on color to convey meaning
